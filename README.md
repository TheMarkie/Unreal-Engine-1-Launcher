# Unreal Engine 1 Launcher
A custom launcher for Unreal Engine 1 games that comes with modern features:
* Borderless Windowed.
* Raw Input.
* FPS Cap.
* Extra mouse keys.
* Misc fixes and features.

# Branches
* **master**: Deus Ex.
* **ut**: Unreal Gold.
